ddb Webapp

Die ddb-Webapp ist das Datenerhebungs-Backend der Datenbank dialogische Beteiligungsverfahren. Die Webanwendung integriert die verschiedenen Arbeitsschritte und verteilt die anfallenden Arbeitspakete.

Datenbestand

Bisher wurden 9999 Beteiligungsereignisse in 9999 Kommunen in Baden-Württemberg identifiziert. Sie können den Datenbestand gerne erkunden. Verschiedene Such- und Filtermöglichkeiten helfen Ihnen zu finden, was Sie interessiert. Fü weitergehende Auswertungen steht ein Scientific Use File zur Verfügung.

Über das Projekt

Die Auswertungen auf dieser Seite basieren auf den Daten der Datenbank dialogorientierte Beteiligunsverfahren (DDB), die im Rahmen von Drittmittelforschungs am Institut für Sozialwissenschaften der Universität Stuttgart gesammelt werden.

Hinter den Kulissen

Eine speziell für das Projekt entwickelte Anwendung integriert die verschiedenen Schritte der Datenerhebung und Aufbereitung. Es handelt sich um eine Webanwendung die in Python auf Basis des Django Webframeworks geschrieben wurde. Eine ausführliche Dokumentation und den Code gibt es auf Github .
